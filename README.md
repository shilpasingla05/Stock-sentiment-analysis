# Stock Market Sentiment Analysis 📉📰

This project analyzes Twitter sentiment and its correlation with stock market movements, using NLP techniques and SPY (S&P 500 ETF) as the market indicator.

---

## 🔍 Project Overview

- **Goal**: Understand whether public sentiment on Twitter correlates with daily SPY closing prices.
- **Dataset**: [Sentiment140](https://www.kaggle.com/datasets/kazanova/sentiment140) (1.6M labeled tweets).
- **Tech Stack**: Python, pandas, scikit-learn, NLTK, matplotlib, yfinance

---

## 🛠️ Setup Instructions

### 1. Clone the Repo
```bash
git clone https://github.com/shilpasingla05/stock-market-sentiment-analysis.git
cd stock-market-sentiment-analysis
``` 
### 2. Setup dataset
Download the dataset from [Sentiment140](https://www.kaggle.com/datasets/kazanova/sentiment140) and place into content folder. Rename the csv file to training.1600000.processed.noemoticon.csv
